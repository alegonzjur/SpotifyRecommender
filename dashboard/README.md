# Dashboard Power BI — Spotify Music Recommender

## Archivos

- `spotify_analysis.pbix` — archivo principal (se genera en la Fase 5)
- `theme_spotify.json` — tema personalizado con paleta de colores del proyecto

## Páginas previstas

1. **Resumen general** — KPIs: total canciones, géneros, artistas, popularidad media
2. **Audio features** — Distribuciones de danceability, energy, valence, tempo por género
3. **Mapa de géneros** — Scatter plot energy vs valence coloreado por género
4. **Evolución temporal** — Tendencias de audio features por año de lanzamiento
5. **Top artistas** — Ranking por popularidad media y número de canciones

## Fuente de datos

El dashboard se conecta a `data/processed/tracks_clean.csv`.
